# Chat local em python

Um chat em python utilizando a interface socket para comunicação entre os clientes.

## Clonando o repositório

```bash
git clone https://github.com/maycondsn/chat-cli-em-python.git
```

## Run

### Rodar o servidor

```bash
python3 server.py
```

### Rodar o cliente

```bash
python3 main.py
```
